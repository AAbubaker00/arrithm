SolveXpress API Documentation

API Paths

  • /api/v1/solve-expression/{latextExpression}

Input Syntax Rules

Here are the input syntax rules:

  • Remove spaces from input.
  • If a backslash "\" is present at the beginning, convert it to "%5C".
  • URL-encode special characters in the expression, such as "&" as "%26".
  • Replace "(" with "{".
  • Replace ")" with "}."
  • Replace "\\" with "`".
  • Replace "\" with "`".
  • Replace "sqrt" with "\sqrt".

Get Started

To get started, use the following example URL path to solve an expression: /api/v1/solve-expression/%5Cint_{1}^{3}4xdx
